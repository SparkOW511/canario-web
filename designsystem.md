1. EcoPulse
Core Philosophy

Recycling as a living ecosystem. Data and sustainability feel organic, dynamic, and alive. The system visualizes recycling as flowing cycles rather than static bins.

Inspired by: Strava, Headspace, Oura

Color Palette

Primary colors simulate natural cycles.

Role	Color	Hex	Psychology
Primary Green	Forest Pulse	#1DB954	growth, sustainability, vitality
Secondary	Leaf Mist	#B8F2D6	calm, natural freshness
Accent	Citrus Spark	#F7B801	energy, reward feedback
Background	Soil Light	#F5F7F6	neutral ecological base
Dark Mode	Deep Forest	#0E2F2A	depth, environmental seriousness
Typography

Primary Font: Inter

Hierarchy:

Level	Size	Weight
Hero Metric	48px	Bold
Section Title	24px	SemiBold
Body	16px	Regular
Data Labels	12px	Medium

Rule:

Metrics always bold
Environmental stats highlighted in accent color
Spacing + Grid

Grid: 12-column responsive

Spacing scale:

4px
8px
16px
24px
40px
64px

Cards use 24px internal padding.

Signature Visual Element

Circular Recycling Rings

Each recycling category (plastic, paper, metal) is represented by a ring that fills up as the user recycles.

Example:

Plastic: blue ring
Paper: beige ring
Glass: green ring

The rings subtly animate like breathing.

Motion Principles
Circular progress flows
Micro-animations mimic natural breathing rhythms
Achievement animation: leaf particles bloom outward

Interaction timing:

Type	Duration
Hover	120ms
Progress update	400ms
Achievement	900ms
